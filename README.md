# SofiCoin 官方网站

融合AI、生理/神经反馈与Psilocybin微量疗法的潜意识重塑闭环平台

## 📁 文件结构

```
SofiCoin-Website/
├── index.html          # 主页面
├── styles.css          # 样式文件
├── script.js           # 交互脚本
├── README.md           # 说明文档
└── assets/             # 资源文件夹（需要创建）
    ├── images/         # 图片
    ├── whitepaper.pdf  # 白皮书PDF
    └── whitepaper-en.pdf # 英文白皮书
```

## 🚀 快速开始

### 方法1：直接打开
1. 双击 `index.html` 文件
2. 网站将在默认浏览器中打开

### 方法2：使用本地服务器（推荐）
```bash
# 使用 Python
cd SofiCoin-Website
python -m http.server 8000

# 使用 Node.js
npx http-server

# 使用 PHP
php -S localhost:8000
```

然后访问：http://localhost:8000

## 🎨 自定义

### 修改颜色主题
在 `styles.css` 文件顶部的 `:root` 部分修改：

```css
:root {
    --primary-color: #6B46C1;    /* 主色调 */
    --secondary-color: #48BB78;  /* 次要色 */
    --accent-color: #F6AD55;     /* 强调色 */
}
```

### 添加新板块
在 `index.html` 中添加新的 `<section>` 标签，并在导航栏添加对应链接。
